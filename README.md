# forwardproxy

### Description

- Ready to use dockerized forward proxy written in golang
- Binds to 0.0.0.0:8080 (not secure! if exposed on machine without firewall)


### How to use it 

Dockerize you image 

``` make dockerize ```

or use ready to go image 

``` docker pull basilboli/forwardproxy ```
